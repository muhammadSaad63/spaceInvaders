# Space Invaders 👾

**Space Invaders** is a high-performance C++ recreation of the famous **Space Invaders** game, built using the **raylib** framework and using **OOP** principles.

---

[Directory Structure]

    spaceInvaders/
    ├── src/                # All .cpp and .hpp files
    │   ├── core/           # Database, Game engine, Background
    │   ├── entities/       # SpaceShip, Alien, Laser, MotherShip
    │   ├── states/         # Menu, Playing, Settings, etc.
    │   └── main.cpp        # Entry point
    ├── assets/             # Graphics, Sounds, Databases
    ├── LICENSE
    └── README.md
